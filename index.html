Desafio de Projeto - Criar uma Página Web Utilizando as Tags Aprendidas

Este é o primeiro desafio de projeto realizado no Bootcamp NTT DATA Diversidade em Tech. A trilha é focada em desenvolvimento Front-End, trazendo HTML, CSS, JavaScript, Angular e AWS. É voltada para grupos minoritários como mulheres, pessoas pretas, indígenas, comunidade LGBTI+, pessoas com deficiência, pessoas +50, entre outros. Neste desafio precisamos realizar uma página Web apenas com as tags HTML aprendidas.

Pessoas importantes para a área da tecnologia

  • Grace Murray Hopper

    - Uma das primeiras programadoras do Mark I em 1944. Criadora da linguagem de programação de alto nível Flow-Matic (base do COBOL)

  • Joseph Carl Robnett Licklider

    - Plantou a "semente" da comunicação entre dois pontos distintos através do computador.

  • Robert E. Kahn

    - Robert E. Kahn - apresentou o primeiro e-mail. Criador do TCP/IP juntamente com Vinton Cerf.

  • Tim Berners-Lee

    - Inventor do WWW. Popularizou o HTTP e HTML. Diretor do World Wide Web Consortion (W3C).

  • Marc Andreessen

    - Reinventou o navegador (Netscape Navigator, feito a partir do Mosaic). Tornou o navegador mais "amigável" para os usuários comuns, trazendo recursos gráficos.

  • Bill Gates

    - Fundador da Microsoft, criadora do Internet Explorer. Tornou o computador acessível para muitos, e com isso popularizou ainda mais o acesso à internet.

  • Steve Jobs

    - Fundador da Apple, concorrente direta da Microsoft. Popularizou o smartphone que conhecemos hoje através do lamçamento do iPhone em 2007.

Comunicação Client x Server

O modelo cliente-servidor é uma estrutura de aplicação distribuída que divide as tarefas entre servidor e cliente. Nele, o servidor é responsável por receber as requisições e enviar os dados requeridos ao cliente, e o cliente é responsável por fazer as requisições para o servidor.
